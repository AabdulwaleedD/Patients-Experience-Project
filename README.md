## Lessons Learned & Takeaways

- Data cleaning takes longer than visualization â€” always start there.
- Excel Tables + structured references prevent a lot of headaches.
- PivotCharts + connected slicers give near-Power-BI interactivity for free.
- Writing "KEY INSIGHTS" on every page forces real business value.
- One focused visual (radar, scatter with trendline) beats many cluttered ones.
